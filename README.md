# マーケット・ファン (Market Fan)

> 株価指数の**較正済み確率ファンチャート** — 1点の予想ではなく「起こりうる分布」を、外した過去も隠さずに示すアプリ。

S&P 500 と日経225 の今後 **3か月〜20年** の値動きを、確率分布(ファンチャート)として毎日自動更新で予測します。EN / 日本語 切替対応。

🔗 公開アプリ: https://sp500-forecast-4xldbbjrzkvbbxpcmhblsp.streamlit.app

---

## このアプリの哲学:正直さ第一

普通の「予測アプリ」と違い、**当たることより、外れ方まで正直に見せること**を最優先しています。

- 予測は**1つの数字ではなく確率分布**(ファン)。「典型はここ、でもこの範囲で外れうる」を示す。
- すべて**ウォークフォワード検証**(各時点で「その時に入手可能なデータだけ」で予測)。後知恵なし。
- **外した過去を消さない**(④答え合わせ・⑤タイムマシン)。2007年→金融危機の −37% も「予見できなかった」とそのまま表示。
- 効かないと分かったものは**「検証して棄却した」と数字で示す**(マクロ指標・VIX・日本のバリュエーション代替は、いずれも1年予測ではシグナル無し)。

> いちばん正直な結論: **1年先の株価はほぼ誰にも当てられない**(⑧ベンチマーク参照)。価値があるのは「較正された不確実性」であって、水晶玉ではありません。

## 9つの画面(タブ)

| タブ | 内容 |
|---|---|
| ① ファンチャート | 中央値+50%/90%帯のファン。下に**リターン分解**(利益成長+バリュエーション変化+配当)と**株 vs 債券 vs 現金** |
| ② 指標ヒートマップ | 16指標(CAPE・イールドカーブ・配当/益利回り・ERP・実質金利・インフレ等)の現在地を z スコアで色分け |
| ③ 較正 | このモデルは正直か? カバー率(50/80/90%)・PITヒストグラム・**VIX精度テスト** |
| ④ 答え合わせログ | 過去の予測 vs 実際の着地。危機の事例(外した例も) |
| ⑤ タイムマシン | 過去の任意の時点を選び、**当時描いたファン vs 1年後の実際**を対比 |
| ⑥ モデルの仕組み | 手法のやさしい解説 |
| ⑦ 試算 | **確率計算機**(X以上/未満の確率)+**シナリオ What-if**(CAPE/ボラ/ショックを動かす)+**ダウンサイドリスク**(VaR・CVaR・弱気確率・債券に負ける確率) |
| ⑧ ベンチマーク | **世界標準モデルと対決**(ランダムウォーク/過去平均/配当・CAPE利回り/Goyal-Welch)。OOS R²・Clark-West・pinball |
| ⑨ 市場比較 | S&P 500 と日経のファンをリターン%で重ね描き |

対象: **S&P 500**(CAPE バリュエーション・アンカー)/ **日経225**(価格のみのモデル・中立ドリフト)。
期間: **3か月・6か月・1年・2年・3年・5年・10年・15年・20年**。

## 仕組み(3層)

1. **中心(ドリフト)= Level 0 構造アンカー**
   対数空間で `ln P̂(t+12) = ln P_t + g + λ·(ln CAPE* − ln CAPE)`。利益成長 `g` で押し上げ、30年中央値バリュエーション `CAPE*` の方向へ *小さく*(λ≈0.07–0.11)引き戻す。**λ=1(完全回帰)は壊滅的**と実証済みなので採らない。
2. **幅 = GARCH(1,1) 条件付きボラティリティ**
   市場が荒れている時はファンが広がり、平穏なら狭まる。
3. **形 = FHS(フィルタード・ヒストリカル・シミュレーション)**
   左右対称の釣鐘でなく、過去のサプライズの実形状を使う。だから**下振れの裾が上振れより太く**、中央値が平均より上にくる。

検証済みの正直な数字(⑧で確認可能):
- **1年**: モデルは過去平均にわずかに勝つが、その大半は EPS 成長で、**CAPEタイミングの増分はほぼゼロ(戦後はむしろ害)**。標準予測子は単体でほぼ勝てず、全部入りは過学習で大敗(= Goyal-Welch 2008)。
- **長期(10年)**: ここで初めて CAPE が効くが、独立した10年窓は約8個しかない(**n_eff 小**)。長期の帯はブートストラップ信頼区間付きで「**較正済みでなく目安**」と明示。

## データと自動化

- **データ**: Shiller(CAPE・利益、1871年〜)、FRED(マクロ14系列)、Yahoo Finance(^GSPC・^N225)
- **毎営業日**: GitHub Actions が データ取得 → 予測再計算 → **信頼性テスト** → JSONコミット → Streamlit Cloud が自動再デプロイ
- **信頼性ゲート(CI)**: 危機6エピソード・恒等式・退化λ・リーク検査・**較正ゲート(cover90∈[0.86,0.94] を外れたらコミットを止める)** など15テストを push/PR と日次で実行
- **週次ダイジェスト**: Discord webhook へ要約投稿(要 secret `DISCORD_WEBHOOK_URL`)

## ローカルでの実行

```bash
uv sync                                  # 環境再現
uv run python fetch_sp500_data.py        # データ取得(当日キャッシュあり)
uv run python forecast.py                # 予測再計算 → app/*.json
uv run python benchmarks.py              # ベンチマーク → app/benchmarks.json
uv run streamlit run app/streamlit_app.py  # アプリ起動
uv run pytest -q                         # 信頼性テスト
```

主要ファイル: `forecast.py`(日次エンジン)/ `benchmarks.py`(対決)/ `app/streamlit_app.py`(UI、計算はせずJSONを読むだけ)/ `phase3_level0.py`・`phase5_fanchart.py`(検証済みモデル)/ `PLAN.md`(モデル仕様・設計判断)/ `tests/`(信頼性ゲート)。

## 正直な限界

- **1年の株式リターンは大部分が予測不能**。どんなバリュエーション/ボラモデルも 2008年を予見できなかった。
- 長期(5〜20年)は独立標本が少なく「目安」。20年は実質1窓。
- **日本版にはバリュエーションの背骨が無い**(無料の長期日本CAPEが存在しない)。価格トレンド代理を検証したが1年シグナル無し(λ≈0)→ コンテキスト表示のみ、**1989年バブルは当てない**。
- 投資助言ではありません。
