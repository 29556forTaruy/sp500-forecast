## サニティチェック(3/3 合格)

1. **SPX 直近終値の実勢照合 — ✅ 合格**
   `spx_daily.csv` の 2026-06-12 終値 **7431.46** を独立2ソースで確認: CNBC クォートAPI(.SPX = 7,431.46、前日終値 7,394.30)、FRED 公式 SP500 系列(2026-06-11 = 7394.30、6/10 = 7266.99 等が日次で完全一致)。
2. **CAPE の Shiller 公表値照合 — ✅ 合格**
   当方 2026-06 暫定 CAPE **41.02**(4月 37.96 / 5月 40.16)に対し、multpl.com 現在値 41.20(6/11時点、月次: 4月 38.93 / 5月 41.10)。乖離は0.2〜1.0ポイントで、multpl が時点価格・Shiller 本家が月中平均価格を使う既知の方法論差の範囲内。
3. **年間ピボットの手計算照合 — ✅ 合格**
   2025年実績 H=6945.77, L=4835.04, C=6845.50 から手計算: P=6208.77, R1=7582.50, S1=5471.77, R2=8319.50, S2=4098.04 → `annual_pivots.csv` の2026年行と完全一致。さらに独立監査エージェントが**全100行×全列を再計算し最大誤差 4.5e-13** を確認。

## 既知の制限・データの癖(対処済み/文書化済み)

| 重要度 | 内容 | 対処 |
|---|---|---|
| **制限** | **BAMLH0A0HYM2(ハイイールドOAS)は直近3年分のみ**(2023-06〜)。キーレス fredgraph.csv がライセンス系列の `cosd/coed` を無視して直近窓しか返さないため | クランプ自動検出を実装。全履歴(1996〜)が必要なら無料 FRED API キーで再取得(フェーズ3で判断)。長期の代替: Moody's BAA−AAA スプレッド |
| 軽微 | spx_daily の 2023-05-24 のみ volume=0(Yahoo 側グリッチ、価格は正常) | 記録のみ(値の捏造はしない) |
| 仕様 | UNRATE 2025-10 欠損 = 政府閉鎖で家計調査が未実施(ソース自体に存在しない) | そのまま保持 |
| 仕様 | UMCSENT は1978年以前が四半期調査 → 月次欠損 ~24% | そのまま保持 |
| 仕様 | Shiller の D/E は公表ラグで直近3ヶ月 NaN。直近月の P は部分月(6/2単日) | フェーズ3のラグ処理で対応予定 |
| 仕様 | ^GSPC は1962年以前全日+1983年6月まで散発38日が High=Low=Close | ピボットの H−L は当該期間「終値レンジ」と解釈 |
| 仕様 | 1928年のピボット行は1927年が1営業日しかなく退化 | モデリングでは1929年以降を使用 |
